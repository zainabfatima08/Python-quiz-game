# Python-quiz-game

Console-based Python quiz game with high score and feedback system

# Features 

- Multiple-choice questions with 4 options each
- Randomized question and option order
- High score tracking (saved in a file)
- Score percentage and feedback (Excellent / Good / Needs Improvement)
- User input validation
- 
# Concepts Covered 

- Variables, loops, and conditionals
- Lists and dictionaries
- Functions and modular programming
- File handling (read/write high scores)
- Exception handling (try-except)
- User input validation
- Randomization using `random.shuffle()`
